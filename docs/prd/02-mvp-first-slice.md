# PRD 第二章：MVP 第一版（首个角色 / 武器 / Build 套）

> 提供大的方向而非详细描述设计。第一章见 `docs/prd/01-core-vision-and-build-system.md`。

---

## 一、美术风格：成熟像素风

整体风格为 **成熟像素风**，像素成熟度参考 **Skul: The Hero Slayer**：高密度像素网格、多级明暗着色、黑色描边、凶悍成熟的剪影。禁止 Q 版 chibi。

约束不变：不引入任何图片素材。像素风通过 PixiJS 程序化实现：

- 用 Graphics 绘制大像素块（例如逻辑像素 1 格 = 屏幕 3~4px 的方块网格）。
- 角色比例：约 3~4 头身，人物在屏幕上足够大、细节足够密，剪影一眼可辨。
- 调色板受限：每个实体 4~6 色，整体统一一套明朝战场色板（明军蓝红、倭寇灰褐、状态色高饱和）。
- 关闭抗锯齿、纹理缩放用最近邻（nearest），保持硬像素边缘。
- 静态像素图形绘制一次后缓存为 RenderTexture 批量复用。
- 动画用少量帧的像素位移/翻转/挤压（squash & stretch），由 ECS 组件驱动。

---

## 二、MVP 目标

一句话：**用一个角色 + 一把武器 + 一条可成型的 Build 线，跑通完整核心循环。**

跑通：进入战斗 → 自动攻击 + 走位 → 击杀获得经验 → 升级三选一兵法 → Build 成型（有明确的"成型瞬间"）→ 存活到时间结束 / 死亡。

MVP 不做：多角色、武器替换、成就解锁、隐藏武器、Boss、局外系统。

---

## 三、首个角色：锦衣卫

- **身份定位**：暴击、背刺、处决。
- **核心机制（MVP 版）**：**处决** —— 对生命低于 20% 的敌人，攻击直接击杀。处决击杀有独立的像素特效反馈。
- **初始武器**：绣春刀。
- **基础属性方向**：中等生命、中上移速（鼓励走位贴近拉扯）、标准拾取半径。
- **像素形象方向**：乌纱帽 + 飞鱼服 + 绣春刀（成熟像素风），主色朱红 + 玄黑 + 金饰边；处决触发时刀光一闪的红黑像素斩击。

---

## 四、首把武器：绣春刀

- **定位**：近战 / 斩击（Slash）。
- **攻击方式**：自动向最近敌人方向挥出扇形刀光，短冷却、中伤害、近距离。
- **玩法感受**：贴身收割——玩家用走位把怪拉成一片，刀光成片斩杀。
- **像素表现**：扇形刀光用 2~3 帧像素弧线，命中敌人白闪 1 帧。
- **与 Build 的关系**：绣春刀是流血与处决的载体；兵法改变它的规则（范围、剑气、附加流血）。

---

## 五、首套 Build：血狱追猎流（MVP 版）

角色（锦衣卫/处决） × 武器（绣春刀） × 兵法（流血） × 状态反应（暂缓） = **血狱追猎流**。

成型路径：

1. 前期拿基础兵法保底（攻速/范围/生命）。
2. 拿到 **见血封喉**（状态兵法：攻击附加流血，持续掉血）。
3. 拿到 **趁其病要其命**（规则兵法：流血中的敌人，处决阈值从 20% 提高到 35%）。
4. 成型瞬间：系统提示 **兵法联动：血狱追猎** —— 流血敌人死亡时，向周围溅射流血。

成型后的体验：刀光过处敌人集体流血，血量压到阈值后连环处决，人群像多米诺一样倒下。

MVP 只需要这一条联动，但"成型提示 + 体验骤变"必须做出来——这是整个游戏乐趣的最小证明。

---

## 六、MVP 兵法池（约 12 个）

按五层比例的最小集合：

**基础（4）**：攻击提升、攻速提升、生命提升、范围提升。

**武器改造（4）**：
- 刀光扩大（扇形角度增大）
- 剑气（挥刀释放短程飞行刀气）
- 连斩（每次攻击挥两刀）
- 拖尾（刀光留下短暂伤害残留）

**状态（2）**：
- 见血封喉（攻击附加流血）
- 血上加血（流血可叠加 2 层）

**规则（2）**：
- 趁其病要其命（流血敌人处决阈值提高）
- 越战越勇（生命越低攻速越快）

核心兵法 MVP 不出现（池太小会失衡），第一次内容扩展时加入。

---

## 七、MVP 敌人与波次

- **倭寇杂兵**：1 种，直线追踪玩家，接触伤害，低血量，死亡掉 XP。像素形象：月代顶髻 + 破旧胴衣 + 打刀，凶悍浪人剪影。
- 波次：10 分钟内密度线性上升的最小曲线；3 分钟后加入"精英杂兵"（同模型放大 + 描边，高血量高 XP）验证 modifier 机制。
- 胜利：存活 10 分钟；失败：HP 归零。

---

## 八、MVP 验收标准

- [ ] 成熟像素风成立：无图片资产，硬像素边缘，人物尺寸与细节密度达到 Skul 级观感，角色/敌人/刀光/XP 一眼可辨。
- [ ] 锦衣卫处决机制生效且有专属反馈。
- [ ] 绣春刀自动攻击 + 四个武器改造兵法都真实改变攻击方式。
- [ ] 流血状态由独立 StatusSystem 结算，敌人身上有流血像素提示。
- [ ] 「血狱追猎」联动可成型，成型有全屏提示，成型前后体验明显不同。
- [ ] 500+ 敌人在屏时帧率稳定。
- [ ] 所有数值与兵法配置在 `src/game/balance/` 可调。
